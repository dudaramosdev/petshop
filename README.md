O QUE É: Um software de petshop é um sistema desenvolvido para gerenciar as principais atividades de uma loja de animais, como o cadastro de clientes, pets, produtos e serviços. Ele facilita o controle de vendas, agendamentos, estoque e atendimentos veterinários, tornando o atendimento mais rápido e organizado. O objetivo é otimizar a administração do petshop, garantindo praticidade e eficiência tanto para os funcionários quanto para os clientes.
COMO RODAR: 
  1 python.exe -m venv venv
  2 .\venv\Scripts\activate
  3 python.exe .\manage.py runserver
